# Contributing

This repository is intended to be a low-maintenance, open research seed.

Useful contributions include:

- reproducible implementations;
- public datasets or synthetic generators;
- evaluation scripts;
- hard-region extraction methods;
- annotation-noise analysis;
- ablation studies;
- negative results and failure cases.

For experiments, please document:

1. Image resolution and defect scale.
2. Annotation type and expected annotation error.
3. Candidate-generation method.
4. Mining thresholds and pool ratios.
5. Patch sizes and context strategy.
6. Defect-level recall and false positives per megapixel.
7. Runtime and hardware.

Do not upload confidential industrial data.
